Privacy Policy

Last updated: February 17, 2026

BotsCleaner ("the Extension") is a browser extension that helps users find and block bots in their Telegram messenger dialogs. Your privacy is important to us. This policy explains what data the Extension accesses and how it is handled.

Data We Access

The Extension accesses the following data from your open web.telegram.org tab:

  • Telegram authentication key (auth_key) — used locally to communicate with the Telegram API on your behalf
  • Data Center ID (dc) — identifies which Telegram server to connect to

How Data Is Used

  • All data is processed locally in your browser. No data is sent to any external server owned or operated by us.
  • The Extension communicates directly with Telegram's official API servers using your existing session — the same servers that web.telegram.org connects to.
  • Your auth_key is used only during the active scan/block session and is not stored permanently by the Extension.

Data We Do NOT Collect

  • We do not collect, store, or transmit your personal data.
  • We do not use analytics, tracking, or any third-party services.
  • We do not have any backend servers that receive your information.
  • We do not read your messages, contacts, or any content other than the list of dialogs and users.

Permissions Explained

  • activeTab — to detect if web.telegram.org is open
  • scripting — to read the authentication key from the Telegram web page (required to use your existing login session)
  • storage — for internal extension state
  • host_permissions (web.telegram.org) — to interact with the Telegram web application only

Third-Party Services

The Extension communicates only with Telegram's official API servers (*.telegram.org). No other third-party services are used.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last updated" date above.
